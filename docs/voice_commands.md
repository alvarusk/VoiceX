# Comandos de voz

- **siguiente / adelante / avanzar / next** → ir a la línea siguiente.
- **anterior / atras / previous** → ir a la línea anterior.
- **duda / marcar duda / quitar duda** → alternar el estado de duda.
- **aceptar / usar / usar voz / confirmar** → usar el texto dictado como selección.
- **rechazar / borrar / limpiar** → borrar el texto dictado.
- **repetir / otra vez / escuchar de nuevo** → volver a escuchar.
- Cualquier otra frase se guarda como texto dictado para la línea actual.
